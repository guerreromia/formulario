10.¿Qué son los métodos / verbos de petición HTTP? Dar ejemplos. Bibliografía MDN

Los métodos de petición son aquellos que se usan para indicar la acción que se desea realizar para un recurso determinado. Aunque estos también pueden ser sustantivos, estos métodos de solicitud a veces son llamados verbos de petición HTTP Ejemplo: request method
